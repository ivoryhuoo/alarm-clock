\doxysubsection{Detailed Description}
Implementation of the \doxylink{class_main_window}{Main\+Window} class for the Rise and Pi alarm clock. 

This file defines the \doxylink{class_main_window}{Main\+Window} class, which serves as the main interface for the alarm clock application. It allows users to set, view, and manage alarms.

The main functionalities include\+:
\begin{DoxyItemize}
\item Displaying the current time.
\item Setting new alarms with labels and sounds.
\item Viewing a list of active alarms.
\item Checking and triggering alarms when the time matches.
\item Snoozing and dismissing alarms via a message box.
\end{DoxyItemize}
